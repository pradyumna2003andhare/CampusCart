# CampusCart

CampusCart is a full-stack marketplace web application designed for college students to buy and sell products within their campus community. The platform allows users to upload products, view listings, and connect with product owners easily.

---

## Features

- User-friendly marketplace interface
- Upload products with images
- Product listing dashboard
- Owner contact details display
- Student-focused buying and selling system
- Responsive design for mobile and desktop
- Backend database integration

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap
- EJS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/campuscart.git
```

Move into the project directory:

```bash
cd campuscart
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
nodemon index.js
```
